# پروژه ساخت پیکره داده‌های سوالات

این پروژه برای ساخت یک پیکره (corpus) از سوالات و پاسخ‌های مربوط به مسائل اعتقادی اسلامی طراحی شده است. هدف این پروژه استخراج سوالات از مقالات و ذخیره آن‌ها به صورت ساختاریافته در قالب JSONL است.
این پروژه برای یک درس دانشگاهی تعریف شده است.

## ویژگی‌ها
- استخراج سوالات و پاسخ‌ها از مقالات دینی اسلامی
- ذخیره‌سازی داده‌ها در فرمت JSONL
- استفاده از ابزارهای مدرن مانند ChatGPT برای استخراج سوالات

## ساختار پروژه
QuestionCorpusProject/
├── data/          # مقالات استخراج شده
├── output/        # خروجی داده‌ها
├── scripts/       # اسکریپت‌های پروژه
├── env/           # محیط مجازی
├── requirements.txt # لیست کتابخانه‌ها
└── README.md      # توضیحات پروژه
